# 项目说明

## 项目概述

**DefectDetection** 是基于 **Qt6 + OpenCV 4.6** 的工业视觉缺陷检测系统，覆盖划痕、裂纹、异物、尺寸偏差四类缺陷，支持产线实时判定。

---

## 功能特性
- 四类缺陷检测，支持多 ROI/多工位。
- 严重度评分：0~100 分，三级判定，阈值可配置。
- 实时节拍：目标 ≤ 100ms/件（采集+处理+判定）。
- 可选 DNN：可挂载 YOLO/MobileNet 模型。
- 产线集成：Modbus/S7/MC 扩展，工业相机（海康/大恒/Basler 等）。
- 质量管理：SPC、报表导出、审计日志。

## 快速开始（qmake）

### 1) 克隆
```bash
# git clone <repo-url>
cd DefectDetection
```

### 2) 构建
- **Windows/MinGW 命令行**：
```powershell
mkdir build; cd build
qmake -r ..\DefectDetection.pro CONFIG+=release -spec win32-g++
mingw32-make -j8
```
- **Qt Creator**：打开 `DefectDetection.pro`，选合适 Kit，构建 Release。
- **Linux**：
```bash
mkdir -p build && cd build
qmake -r ../DefectDetection.pro CONFIG+=release
make -j$(nproc)
```

### 3) 运行
```bash
# Windows Release 例子
build\Desktop_Qt_6_8_1_MinGW_64_bit-Release\src\app\DefectDetection.exe

# Linux
./build/<Kit>/src/app/DefectDetection
```

> 提示：当前仓库无 CMake 构建、无打包/安装脚本，部署前需手工收集 Qt/OpenCV/SDK 运行库（Windows 可用 `windeployqt`）。
