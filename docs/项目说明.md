# 项目说明

## 项目概述

**DefectDetection** 是一套基于 **Qt6 + OpenCV 4.6** 构建的工业视觉缺陷检测系统，支持流水线产品的实时检测与质量判定。

---

## 功能特性

### 核心功能

- ✅ **四类缺陷检测**：划痕、裂纹、异物、尺寸偏差
- ✅ **严重度评分**：0~100 量化评分 + 轻微/中等/严重三级判定
- ✅ **实时检测**：≤ 100ms/件检测节拍
- ✅ **深度学习支持**：可选 YOLO/MobileNet 模型增强检测能力

### 产线集成

- ✅ **PLC 通信**：Modbus TCP/RTU、西门子 S7、三菱 MC 协议
- ✅ **工业相机**：海康、大恒、Basler 等 GigE/USB 相机支持
- ✅ **MES 对接**：REST API 数据上报

### 质量管理

- ✅ **SPC 统计**：控制图、CPK 过程能力分析
- ✅ **追溯报表**：日报/周报/月报自动生成
- ✅ **审计日志**：操作记录完整追溯

## 快速开始

### 1. 克隆仓库

```bash
git clone [https://github.com/your-org/defect-detection.git](https://github.com/your-org/defect-detection.git)
cd defect-detection
```

### 2. 安装依赖

```bash
# Ubuntu
sudo apt install qt6-base-dev libopencv-dev

# Windows (vcpkg)
vcpkg install qt6 opencv4
```

### 3. 构建项目

```bash
mkdir build && cd build
cmake ..
cmake --build . --config Release -j8
```

### 4. 运行程序

```bash
./bin/defect-detection
```
